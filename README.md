# Server

TCP сервер для распределённого вычисления интеграла. Принимает подключения 
клиентов, распределяет задачу пропорционально количеству ядер CPU каждого 
клиента и собирает результаты.

## Зависимости

- CMake 3.20+
- GCC / MinGW
- Boost 1.90+

## Сборка и запуск

Классический для CMake, но есть неприятный момент в том, что у меня
мало опыта рапсространения проектов. И в CMakeLists.txt и CMakePresets.json
жестко указаны пути к компиляторы, библиотеке Boost, их нужно поменять на свои / убрать вовсе.
